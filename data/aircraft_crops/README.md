# Aircraft Crops

Put labelled aircraft crop images in this folder.

Suggested naming:

```text
GZOFG_P28A_001.jpg
MGETS_PC12_001.jpg
EDY01_A109_001.jpg
unknown_airplane_001.jpg
```

Use tight crops around the aircraft, not full webcam frames. These files are
ignored by git so local examples are not uploaded to GitHub.
